# Netflix-Movies-TV-Shows-Dashboard-using-Tableau

Project Overview

This project is an interactive data analysis dashboard built using the Netflix Movies & TV Shows dataset. The goal of this project is to analyze Netflix content based on type, country, genre, ratings, and yearly trends, and present clear business insights using visualizations.

This dashboard helps answer questions like:

What type of content does Netflix offer more – Movies or TV Shows?

Which countries contribute the most content?

What are the most common content ratings?

Which genres dominate Netflix?

How has Netflix content grown over the years?


Key Columns Used
Column Name               Description
show_id -            Unique ID for each title
type -               Movie or TV Show
title -              Name of the content
director -         	 Director of the title
cast -               Main actors
country -            Country where the content was produced
date_added -         Date when content was added to Netflix
release_year -       Original release year
rating -             Content rating (TV-MA, TV-14, etc.)

2️. Total Movies & TV Shows by Country (World Map)
Shows global distribution of Netflix content
Darker color = higher number of titles
Key Insights:
United States contributes the highest number of titles
India, UK, Canada, and Japan are major contributors
Netflix focuses heavily on international content expansion

3️. Ratings Distribution (Bar Chart)
Displays count of titles by content rating
Key Insights:
TV-MA has the highest number of titles → Netflix focuses on mature content
TV-14 is the second most common rating
Very limited content for kids (G, TV-Y) compared to adult content
